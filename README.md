# Quiz-App
A Simple quiz where user can answer simple questions 
and finally checks last score and user details are stored in firebase.

External Packages used:
1.Firebase Auth
2.Firebase Core
3.rFlutter Alert
4.Modal Progress Hud
5.Animated Text Kit
6.Cloud Firestore
